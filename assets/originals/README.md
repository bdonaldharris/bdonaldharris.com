# Original image masters

High-resolution source photos kept out of `public/` so they are **not served or
deployed**. Web-optimized derivatives in `public/images/` are what the site uses:

- `profile-photo-one.jpg` → `public/images/profile-photo-one-web.jpg`
- `profile-photo-two.png` → `public/images/profile-photo-two-web.jpg`

Re-export with `sips` (or any image tool) if a new crop/size is needed.
